# 📊 Global Superstore Analysis & Business Intelligence Dashboard

## 📌 Project Overview

The **Global Superstore Analysis & Business Intelligence Dashboard** is an end-to-end Data Analytics project that transforms raw retail sales data into meaningful business insights using Python, Machine Learning, and interactive Dash visualizations.

The project performs data preprocessing, exploratory data analysis (EDA), KPI generation, customer segmentation, sales forecasting, and presents the results through an interactive dashboard that helps stakeholders make data-driven decisions.

---

# 🚀 Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Interactive Business Dashboard using Dash & Plotly
* Sales Performance Analysis
* Profit Analysis
* Customer Segmentation
* Regional Sales Analysis
* Category & Sub-Category Analysis
* Monthly Sales Trend Analysis
* Sales Forecasting
* KPI Cards
* Interactive Filters
* Business Insights Report

# ⚙️ Installation

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/global_superstore_analysis.git
```

## 2. Navigate to the Project

```bash
cd global_superstore_analysis
```

## 3. Create a Virtual Environment

Windows

```bash
python -m venv venv
```

Activate the virtual environment

```bash
venv\Scripts\activate
```

## 4. Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Running the Dashboard

From the project directory, run:

```bash
python backend/app.py
```

The dashboard will start locally and can be accessed in your web browser.
